Usually, by default our app will be hosted at:

> http://localhost:3000

However, for this session, click on the _Dashboard_ tab and enter **3000** into the port.  
Click on **Display port** to see the default starter app.

If you had created your react project on your actual machine (rather than this in-browser demo), as soon as you start your development server it will open a new tab in your browser automatically at the above URL.

The code to create this UI is what comes with this starter project. Press Continue to take a look.
